# Tasks Tracker

A modern task tracking web app built with Vue.js.

## Features

- Add and remove tasks
- Mark tasks as completed
- Filter tasks
- Persistent storage using localStorage
- Responsive card-based UI
- Interactive hover effects

## Tech Stack

- Vue 3
- JavaScript
- HTML
- CSS
